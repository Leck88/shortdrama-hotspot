# shortdrama-hotspot

**短剧热点监控工具** —— 每日自动抓取短剧热度排行榜、抖音短剧相关热搜，生成结构化日报。

> 适用于短剧创作者、选题策划、行业研究者，快速掌握短剧市场热点与题材趋势。

## 功能

- **短剧热度榜 Top30** —— 调用酷乐免费API，获取当日最新短剧热度排名
- **抖音热搜智能筛选** —— 从抖音热搜中自动筛选短剧/影视相关词条（支持两级关键词匹配 + 误匹配过滤）
- **题材自动分类** —— 根据剧名关键词，将短剧归类为婚恋、霸总、甜宠、逆袭等10种题材
- **结构化日报生成** —— 输出Markdown格式报告，包含热度榜、抖音热搜、题材分布、选题参考
- **定时自动化支持** —— 可配合cron/任务计划/WorkBuddy定时执行，每日自动生成报告

## 数据源

| 数据源 | 接口地址 | 说明 |
|--------|----------|------|
| 短剧热度榜 | `https://api.kuleu.com/api/shortdramarank` | 免费，无需Key，每日更新30条排名 |
| 抖音热搜 | `https://v2.xxapi.cn/api/douyinhot` | 免费，无需Key，实时更新 |

两个API均为免费公开接口，无需注册和鉴权，直接调用即可。

## 安装

无需安装任何第三方依赖，使用 Python 3 标准库（`urllib`, `json`）即可运行。

```bash
git clone https://github.com/Leck88/shortdrama-hotspot.git
cd shortdrama-hotspot
```

## 使用方式

### 快速查看（控制台输出）

```bash
python fetch_hotspot.py
```

在控制台输出短剧热度摘要，并在末尾输出JSON格式完整数据（供程序集成使用）。

### 生成日报文件

```bash
python fetch_hotspot.py --report --output "D:/视频生产/reports/shortdrama"
```

生成 `短剧热点日报_YYYY-MM-DD.md` 文件到指定目录。不指定 `--output` 则使用默认目录。

### 仅获取热度榜（跳过抖音热搜）

```bash
python fetch_hotspot.py --rank-only
```

适用于只需要短剧排名、不需要抖音热搜数据的场景，执行更快。

### 参数说明

| 参数 | 说明 |
|------|------|
| `--report` | 生成Markdown日报文件并保存到本地 |
| `--output <目录>` | 指定报告输出目录（默认 `D:\视频生产\reports\shortdrama`） |
| `--rank-only` | 仅获取短剧热度榜，跳过抖音热搜获取 |

## 日报格式

生成的Markdown日报包含以下板块：

```markdown

| 排名 | 剧名 | 热度 |
|------|------|------|

（可折叠展开完整Top30）

## 📱 抖音短剧相关热搜

## 💡 选题参考
基于今日热度数据的分析建议（最热题材、头部剧目、差异化机会等）
```

## 题材分类说明

脚本根据剧名中的关键词自动分类，支持以下10种题材：

| 题材 | 匹配关键词示例 | 说明 |
|------|---------------|------|
| 霸总 | 总裁、首富、豪门 | 权力型男主 |
| 婚恋 | 婚、妻、闪婚、领证 | 婚姻恋爱关系 |
| 甜宠 | 甜、宠、恋、上瘾 | 甜蜜宠爱向 |
| 逆袭 | 逆袭、巅峰、无敌 | 弱者翻盘崛起 |
| 重生 | 重生、穿越、八零 | 回到过去重新来过 |
| 古装 | 皇、帝、妃、太子 | 古代/宫廷背景 |
| 复仇 | 复仇、恩断 | 报仇雪恨 |
| 悬疑 | 案、侦探、真相 | 案件悬疑推理 |
| 战神 | 战龙、修仙、枭雄 | 战斗/武力型 |
| 逆袭/翻盘 | 离婚、出狱、撤资 | 低谷后翻盘 |

未匹配任何关键词的归入"其他"类别。一部剧可能同时属于多个题材。

## 抖音热搜筛选逻辑

从抖音热搜中筛选短剧相关词条采用两级匹配策略：

1. **精确匹配**：词条包含"短剧""微短剧""竖屏剧"等直接关键词 → 优先返回
2. **泛化匹配**：词条包含"剧""开播""追剧""霸总"等影视相关关键词 → 补充返回
3. **误匹配过滤**：排除"剧毒""剧烈""剧增"等含"剧"但非剧集含义的词条

## 定时自动化

### 配合 WorkBuddy

在 WorkBuddy 中创建自动化任务，每日09:00自动执行：

```
python fetch_hotspot.py --report --output "D:/视频生产/reports/shortdrama"
```

### 配合 cron（Linux/macOS）

```bash
# 每天早上9点执行
0 9 * * * cd /path/to/shortdrama-hotspot && python3 fetch_hotspot.py --report
```

### 配合任务计划程序（Windows）

创建基本任务，触发器选"每天"，操作选"启动程序"，参数填 `--report --output "D:/reports"`。

## 输出格式说明

### 控制台模式（不加 --report）

1. 打印数据摘要（Top5热度榜 + 抖音热搜 + 题材分布）
2. 输出 `---JSON_OUTPUT---` 分隔线后的JSON数据，包含完整的热度榜、热搜、题材统计和报告内容，便于其他程序解析集成

### 文件模式（加 --report）

在指定目录生成 `短剧热点日报_日期.md` 文件，可直接用Markdown编辑器或GitHub查看。

## 注意事项

- 短剧热度榜API为免费第三方接口，可能偶尔不稳定或数据更新延迟
- 抖音热搜为综合热搜，短剧相关词条数量取决于当日热搜内容
- 热度值为相对值，不同平台的数值不可直接对比
- 题材分类基于剧名关键词匹配，无法识别剧名中未体现的题材
